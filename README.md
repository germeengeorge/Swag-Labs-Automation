# Swag Labs Automation Testing Project

## Overview
This project is an automated testing framework for the Swag Labs web application using Java and Selenium WebDriver.

The framework is designed using:

- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Fluent Page Object Pattern

The project includes automated test scenarios covering major functionalities of the application such as:

- Login
- Product Listing
- Product Filter
- Add to Cart
- Remove from Cart
- Checkout Process
- Validation & Negative Scenarios

---

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Fluent Pattern Approach
